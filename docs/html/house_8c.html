◆ house_compare()

int house_compare ( const void * h1,
const void * h2 )

Функция-компаратор: сравнивает дома по шести параметрам в следующем порядке:

  1. Год постройки (чем больше год, тем лучше)
  2. Наличие лифта
  3. Тип (кирпичный > монолитный > панельный)
  4. Средняя площадь квартир (чем больше, тем лучше)
  5. Количество квартир (чем больше, тем лучше)
  6. Количество этажей (чем больше, тем лучше)
Аргументы
h1указатель на первый дом
h2указатель на второй дом
Возвращает
int отрицательное число, если h1 < h2, положительное, если h1 > h2, 0, если h1 = h2
