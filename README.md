# Лабораторная работа №3 "Многоквартирный дом"
Программа работает в 3 режимах: генерация тестовых дан-
ных, сортировка данных, печать данных на экран. Режим работы программы выбирается пользователем с помощью ар-
гументов командной строки.
1) Программа, запущенная с флагом --generate N или -g N, генерирует
случайные данные в соответствии со схемой данных в  количестве N строк.
- Если указан флаг --out="output.csv" или -o output.csv, вывод осу-
ществляется в соответствующий файл а формате значений, разделен-
ных запятой (CSV). Иначе вывод осуществляется на стандартный по-
ток вывода в соответствующем формате.
- Если указан флаг --in="data.csv"или -i data.csv, ввод осуществля-
ется из соответствующего файла. Иначе ввод осуществляется со стан-
дартного потока ввода в соответствующем формате.
По-умолчанию сортировка осуществляется по возрастанию. Также по 
возрастанию следует сортировать, если указан флаг --type=ascили -t 
A. Если указан флаг --type=desc или -t D, сортировка выполняется в 
порядке убывания.
2) Программа, запущенная со флагом --sort или -s, считывает данные 
в соответствии со схемой данных в структуру-контейнер (см. вариант), 
сортирует их и выводит отсортированные.
- Если указан флаг --out="output.csv" или -o output.csv, вывод осу-
ществляется в соответствующий файл а формате значений, разделен-
ных запятой (CSV). Иначе вывод осуществляется на стандартный по-
ток вывода в соответствующем формате.
- Если указан флаг --in="data.csv"или -i data.csv, ввод осуществля-
ется из соответствующего файла. Иначе ввод осуществляется со стан-
дартного потока ввода в соответствующем формате.
По-умолчанию сортировка осуществляется по возрастанию. Также по 
возрастанию следует сортировать, если указан флаг --type=ascили -t 
A. Если указан флаг --type=desc или -t D, сортировка выполняется в 
порядке убывания.
3) Программа, запущенная с флагом –print или -P, считывает данные и
выводит их в формате таблицы с фиксированной шириной.
- Если указан флаг --in="data.csv"или -i data.csv, ввод осуществля-
ется из соответствующего файла. Иначе со стандартного потока ввода
считывается имя файла, содержащее данные для считывания.
- Если указан флаг --out="output.txt" или -o output.txt, вывод осу-
ществляется в соответствующий файл. Иначе вывод осуществляется на 
стандартный поток вывода.

## Документация
Документация создана при помощи системы Doxygen и содержится в папке docs

## Описание коммитов 
feat - добавление нового функционала
docs - изменение/добавление документации
fix - исправление ошибок
chore - вспомогательные изменения, не влияющие на код и документация
refactor - правки кода без исправления ошибок
